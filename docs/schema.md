## Conventions

- All primary keys are `uuid` — use `add :id, :binary_id, primary_key: true`
- Timestamps use microsecond precision: `timestamps(type: :utc_datetime_usec)`
- Soft-delete pattern: add a `deleted_at utc_datetime_usec nullable` column;
  never hard-delete rows that are referenced by other tables
- Index naming: `<table>_<col(s)>_index` (Ecto default)
- Foreign key naming: `<referenced_table_singular>_id`
